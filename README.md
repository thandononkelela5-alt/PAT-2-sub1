# Morse Code Research

# Overview
Morse code is a method of communication that uses a series of dots (.) and dashes (-) to represent letters, numbers, and punctuation. It was widely used in telecommunication, especially in early radio and telegraph systems.

# Historical Background
Morse code was developed in the 1830s and 1840s by Samuel Morse and Alfred Vail. It was created to send messages over long distances using the telegraph. Before modern communication systems, Morse code played a crucial role in military operations, maritime communication, and emergency signaling.

# How Morse Code Works
Morse code represents each letter or number using a unique combination of:
- Dots (.) → short signals
- Dashes (-) → long signals

# Timing Rules:
- A dot is one unit of time
- A dash is three units of time
- Space between parts of the same letter = 1 unit
- Space between letters = 3 units
- Space between words = 7 units

# Importance of Morse Code
Morse code is still used today in:
- Emergency communication
- Aviation and navigation
- Amateur radio (ham radio)
